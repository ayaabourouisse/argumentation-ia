Ce projet consiste à implémenter un système d’argumentation en intelligence artificielle.
On représente les arguments sous forme d’un graphe, où chaque nœud est un argument et chaque flèche représente une attaque entre arguments.

L’objectif est de déterminer quels arguments sont acceptables, en analysant les conflits et les relations d’attaque.

Pour cela, l’algorithme :

génère toutes les combinaisons possibles d’arguments,
élimine celles qui sont en conflit,
vérifie si les arguments sont défendus,
puis calcule les différentes solutions appelées sémantiques (complete, grounded, preferred).

 Ce projet permet de comprendre comment l’IA peut modéliser des situations de conflit et prendre des décisions de manière logique.
